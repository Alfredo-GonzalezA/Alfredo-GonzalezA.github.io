Facial Recognition System

Developed a facial recognition program in Python using Jupyter Notebook. Implemented the Support Vector Machine algorithm to distinguish between individual faces accurately. Use of Principal Component Analysis was used to help reduce the number of features to get a swifter response from the training. Data representation was used in order to distinguish between two photos. Finally the area under the curve shows us how accurate our model is.
